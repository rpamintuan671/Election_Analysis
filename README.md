# Overview of Election Audit: 
The purpose of this election audit is to figure out the voter turnout for each county, percentage of votes from each county out of the total count and the county with the highest turnout.

## Election-Audit Summary: 
This script can easily be modified to cities within a county and obtain percentages for each candidates. Of course it would also depend on the data used if each voters indicate which cities they are cast their votes. Counties can just be replaced with cities in the script and we would have a more detailed results.
